# Archived SLURM Material

`deprecated/slurm/` stores retired launchers, logs, and rerun-specific jobfiles that are no longer part of the live benchmark path.

- `strict_parity_exp23/generated/demo/`: completed Exp2/Exp3 strict-parity rerun sheets kept only for audit.
- `logs/`: older SLURM stdout/stderr bundles.
- root `run_*.sh` helpers: older one-off submission scripts.

Use `slurm/` for live launchers.
